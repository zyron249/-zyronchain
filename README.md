# ZyronChain

ZyronChain is a simple blockchain project built with Python and Flask.

## Live Explorer

https://zyronchain.onrender.com

## Features

- Proof-of-Work mining
- Genesis Block
- Transactions
- Wallet address generation
- Mining rewards
- Flask REST API
- Web Blockchain Explorer
- GitHub Actions CI
- Render deployment
